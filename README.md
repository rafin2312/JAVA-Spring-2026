## 📖 Course Overview
This course introduces students to the paradigm of object-oriented programming (OOP) utilizing the Java programming language. Emphasis is placed on writing error-free, reusable code and developing software applications to solve real-life problems. Students will learn to understand user requirements, build Graphical User Interfaces (GUI), connect to databases, and properly document their code.

### 🎯 Key Learning Outcomes
* Master the four pillars of object-oriented programming (Encapsulation, Inheritance, Polymorphism, Abstraction).
* Utilize IDEs effectively (Eclipse/NetBeans).
* Develop rigorous debugging and code-tracing skills.
* Create GUI applications using **JavaFX**.
* Implement database connectivity using **JDBC and MySQL**.

---

## 📚 Topics Covered

This repository contains code samples, exercises, and project files for the following core topics:
* **Java Fundamentals:** Data Types, Conditional Statements, Loops, Operators.
* **OOP Concepts:** Classes, Objects, Methods, Constructors, Interfaces, Packages.
* **Data Structures:** Arrays, ArrayList, HashSet, HashMap.
* **File I/O:** Reading, writing, and exception handling.
* **GUI Development (JavaFX):** Labels, Buttons, ChoiceBox, TableView, Scene Shifting.
* **Database (JDBC):** MySQL integration, running SQL queries, Singleton design pattern.
* **Advanced Topics:** Enums, Lambda Expressions, Threads & Multithreading.
